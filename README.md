# ECE450 Assistant Agent

This repository is for the ECE4500J capstone project:

**A Multi-Channel Laboratory and Course Project Assistant Agent**

## Project Goal

The project aims to build an assistant agent that can connect with course or laboratory tools, help answer project-related questions, and support team work across different channels such as APIs and collaboration platforms.

## Current Stage

The team is preparing for the kickoff meeting and Design Review 1 (DR1).

Current work mainly includes:

- Understanding the course requirements and project background
- Researching existing open-source agent projects
- Comparing available LLM APIs
- Checking Feishu open APIs for external agent integration
- Preparing DR1 presentation materials and written report content
